# create-float

Scaffold new Float.js applications with a single command.

## Usage

```bash
# npm
npm create float@latest

# pnpm
pnpm create float

# yarn
yarn create float
```

## Templates

- **basic** - Simple React app to get started quickly
- **full** - Full-featured app with file-based routing
- **api** - API-focused template with backend routes
