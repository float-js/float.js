# test-welcome-screen

Built with [Float.js](https://github.com/float/float.js) - Ultra Modern Web Framework

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Start production server
pnpm start
```
