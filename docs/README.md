# Float.js Documentation

This is the documentation site for Float.js, built with [Next.js](https://nextjs.org) and [Nextra](https://nextra.site).

## Development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the docs.

## Building

```bash
pnpm build
```

## Deployment

The documentation can be deployed to:
- Vercel (recommended)
- Netlify
- GitHub Pages
- Any static hosting service
