## Quick Start

Get started with Float.js in seconds:

```bash
# Create a new project
pnpm create float my-app

# Navigate to your project
cd my-app

# Install dependencies
pnpm install

# Start development server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to see your app.

## Why Float.js?

### Performance First

Float.js is built from the ground up for speed. With esbuild at its core, you get:
- Sub-second cold starts
- 50ms hot module replacement
- Instant production builds

### Modern React

Float.js embraces React 18's latest features:
- Server-side streaming for faster time-to-first-byte
- Automatic code splitting
- Concurrent rendering

### Developer Experience

Focus on building, not configuring:
- Zero-config TypeScript
- File-based routing that just works
- Intelligent error messages
- Built-in development tools
